# ZAVIR Portfolio Website

A modern, responsive, and high-performance static portfolio website for **ZAVIR**. This project showcases the company’s services, core values, and professional achievements.

## 🚀 Live Demo
You can view the live site here: **https://zavir.netlify.app**

## ✨ Features
*   **Fully Responsive:** Optimized for desktop, tablet, and mobile devices.
*   **Modern UI/UX:** Clean, professional design focused on user engagement.
*   **Fast Loading:** Built as a static site for lightning-fast performance.
*   **Contact Integration:** Includes a functional contact section/form.
*   **Service Highlights:** Dedicated sections for core business offerings.

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure for SEO.
*   **CSS3:** Custom styling (or mention Bootstrap/Tailwind if used).
*   **JavaScript:** Interactive elements and animations.

## 📂 Project Structure
```text
├── assets/          # Images, logos, and icons
├── css/             # Stylesheets
├── js/              # JavaScript files
├── index.html       # Main landing page
└── README.md        # Project documentation
```

## 🛠️ Installation & Local Setup
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/faridahmadkan/ZAVIR-Company-website
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd ZAVIR-Company-website
    ```
3.  **Open the site:**
    Simply open `index.html` in your preferred web browser.
